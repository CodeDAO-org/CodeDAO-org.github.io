Layout Principles

Grid & Containers

Max-width: 1200px (dashboard content) Padding: 24px (desktop), 16px (mobile) Grid gap: 24px (cards), 16px (mobile) Border radius: 8px (cards), 6px (buttons/inputs)

Visual Hierarchy

  1. Page title + navigation
  2. Status indicators
  3. Key metrics/stats
  4. Activity feeds
  5. Secondary actions

Do's and Don'ts

✅ DO
  • Use system fonts
  • Minimal colors (grays + one accent)
  • Subtle hover states
  • 8px spacing increments
  • Clear information hierarchy
  • Developer-focused language
❌ DON'T
  • Bright gradients or neon colors
  • Heavy rounded corners (>8px)
  • Gaming/cartoon aesthetics
  • Excessive animations
  • Cluttered layouts
  • Hype-driven copy
