👋 Welcome to CodeDAO

Earn real CODE tokens every time you write quality code. Here’s how to start:

🔗 Step 1: Connect Your Wallet

Go to the Wallet Dashboard and click "Connect Wallet".

  • ✅ MetaMask, Coinbase, WalletConnect supported
  • ✅ Make sure you're on the Base network (Chain ID: 8453)
  • 🆕 New to MetaMask? Download it here

🧩 Step 2: Install the VS Code Extension

Track your commits and earn rewards automatically:

git clone https://github.com/CodeDAO-org/codedao-extension.git
cd codedao-extension && npm install
code . # Press F5 to launch Extension Dev Host

⚠️ Note: Extension is still in dev mode. Marketplace launch coming soon.

💻 Step 3: Write & Commit Quality Code

Make regular commits to your own repo — we’ll track line count and score quality.

You’ll see popups like: ✅ 4.03 CODE earned!

🎁 Step 4: Claim Your CODE Tokens

Visit your Wallet Dashboard and click "Claim" to receive CODE.

💸 Your tokens will be sent to your connected wallet.
