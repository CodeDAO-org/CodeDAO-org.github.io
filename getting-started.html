🚀 Get Started with CodeDAO

Follow this step-by-step guide to start earning CODE tokens for your code contributions in under 10 minutes.

1

Install VS Code Extension

Clone and run the CodeDAO VS Code extension. This extension monitors your coding activity and calculates rewards automatically.

# Clone the CodeDAO extension repository git clone https://github.com/CodeDAO-org/codedao-extension.git # Navigate to the directory and install dependencies cd codedao-extension npm install # Open in VS Code code . # Press F5 to launch Extension Development Host
✅ Success!
You should see a new VS Code window open with "Extension Development Host" in the title. The CodeDAO extension is now running and monitoring your activity.
2

Connect Your Wallet

Connect a Web3 wallet to the Base network where your CODE token rewards will be sent.

  1. Install MetaMask, Coinbase Wallet, or another Web3 wallet
  2. Add the Base network to your wallet
  3. Visit the Wallet Dashboard
  4. Click "Connect Wallet" and select your wallet
⚠️ Base Network Setup
If you haven't added Base network to MetaMask, you can add it automatically at bridge.base.org
3

Start Coding & Earning

Now you're ready to start earning! Write code in any Git repository and commit your changes.

# Create a test project (or use an existing one) mkdir my-codedao-project cd my-codedao-project git init # Create a simple JavaScript file with quality bonuses cat > app.js << 'EOF' /** * A simple calculator with comprehensive documentation * This demonstrates good coding practices for CodeDAO rewards */ class Calculator { /** * Add two numbers together * @param {number} a - First number * @param {number} b - Second number * @returns {number} Sum of a and b */ add(a, b) { return a + b; } /** * Subtract b from a * @param {number} a - First number * @param {number} b - Second number to subtract * @returns {number} Difference of a and b */ subtract(a, b) { return a - b; } } module.exports = Calculator; EOF # Create test file for bonus points cat > app.test.js << 'EOF' const Calculator = require('./app'); describe('Calculator', () => { let calc; beforeEach(() => { calc = new Calculator(); }); test('should add two numbers correctly', () => { expect(calc.add(2, 3)).toBe(5); expect(calc.add(-1, 1)).toBe(0); }); test('should subtract two numbers correctly', () => { expect(calc.subtract(5, 3)).toBe(2); expect(calc.subtract(1, 1)).toBe(0); }); }); EOF # Commit your changes to trigger reward calculation git add . git commit -m "Add Calculator class with comprehensive tests and documentation - Implements basic arithmetic operations - Includes JSDoc documentation for all methods - Comprehensive test coverage - Follows best practices for maintainability"
🎉 Expected Result
You should see a notification popup in VS Code showing something like: "✅ 45 lines × 0.1 × 1.4 = 6.3 CODE earned! 🎉 Tests (+20%), Documentation (+10%), Multi-file (+10%)"
4

Claim Your Rewards

Visit the Wallet Dashboard to see your accumulated rewards and claim them to your wallet.

  1. Go to the Wallet Dashboard
  2. Check your "Pending Rewards" section
  3. Click the "💎 Claim Rewards" button
  4. Enjoy the confetti celebration! 🎊
  5. Add CODE token to MetaMask for easy tracking
🎊 Congratulations!
You've successfully earned and claimed your first CODE tokens! The tokens are now in your wallet on the Base network.

💡 Pro Tips for Maximum Earnings

✅
Always Include Tests
Test files give you a 20% bonus! Create .test.js, .spec.js files or use __tests__ directories.
📝
Document Your Code
Comments that make up >15% of your code earn a 10% bonus. Use JSDoc, inline comments, and README files.
🔄
Commit Regularly
Multiple commits per day earn you an "active contributor" bonus. Break large changes into logical commits.
📁
Multi-file Changes
Modifying 3+ files in a single commit shows you're making substantial contributions worth a 10% bonus.
